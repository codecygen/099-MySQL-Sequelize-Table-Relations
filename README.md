# Sequelize in MySQL
- This is a project which utilizes sequelize in NodeJS to interact with MySQL.
- ".env" file is created for the project. The content is given down below.

```javascript
DB="mysql"
SCHEMA="website"
ADMIN="root"
PASS="NEWPASSWORD"
HOST="localhost"
PORT=3306
```
